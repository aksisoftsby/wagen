Tool ini membantu untuk menggenerator Blast untuk Nomer baru maupun nomor yang sudah ada. Kenapa ? Karena Whatsapp tidak memfasilitasi kita membuat chat baru baik di aplikasi desktop/web/mobile sebelum kita menyimpan kontak tersebut. Sekaligus tool ini akan aman karena verifikasinya manual dan kita klik manual.

Setelah kamu mendapatkan Outputnya, maka kamu bisa mempaste di aplikasi whatsapp kamu dengan mempaste ke nomor kamu sendiri. Untuk membuat chatting ke nomor kamu sendiri gunakan web browser kemudian akses / buka laman http://wa.me/[nomorkamu] ganti [nomorkamu] dengan nomor handphone mu awali dengan 62 untuk wilayah indonesia
